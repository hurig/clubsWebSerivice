REST API webszolgáltatásai

Github repository: https://github.com/hurig/clubsWebSerivice/tree/master

Metódus URL Szolgáltatás
GET /categories Kategóriák lisjáta
GET /categories/{id} Egy kategória
GET /categories/{id}/clubs Egy kategória szórakozó helyei
POST /categories Új kategróia létrehozása
PATCH /categories/{id}/{agelimit} Kategória módosítása (alsó korhatár)
DELETE /categories/{id} Kategória törlése
GET /clubs Minden klub
GET /clubs/{id} Egy szórakozóhely
POST /clubs Új szórakozóhely felvitele
PATCH /clubs/{id}/{guests}/{category} Szórakozóhely módosítása (vendégek száma, kategória)
PUT /clubs/{id} Szórakozóhely módosítása (bármilyen adat)
DELETE /clubs/{id} Szórakozóhely törlése
